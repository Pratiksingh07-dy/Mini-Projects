# CLI To-Do List Manager

A simple command-line To-Do List application built using Python.

## Features

- Add Tasks
- View Tasks
- Update Tasks
- Delete Tasks
- Persistent Storage using JSON

## Run Commands

### Add Task

```bash
python todo.py add "Complete SQL Course"
```

### List Tasks

```bash
python todo.py list
```

### Update Task

```bash
python todo.py update 1 "Complete DBMS Notes"
```

### Delete Task

```bash
python todo.py delete 1
```

## Tech Stack

- Python
- JSON
- argparse
- File Handling
